# Ruff Application

## Devices

- BUtton
- LCD1602
- DHT11 (关闭)
- SR501 (关闭)
- proximity

## Features

- Temperature and humidity realtime monitor
- Upload sensor data on press button
- Use [Menu](https://rap.ruff.io/raps/menu) and [Promise](https://rap.ruff.io/raps/promise) modules

## Usage

```bash
# 安装依赖
rap install
# 排线
rap layout --visual
# 同一局域网, 部署，启动
rap deploy -s
```
